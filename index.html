Game instructions:

Control:

  • In this game you control the red player.
  • Use the arrow keys to move left, up, right and down.
  • Use spacebar to remain in place.
  • Both players moves synchronously; that is, each time you (the red player) make a move, the blue player will make a move as well---at the same time.

Goal:

Check this with Amos!

Your goal is to collect all awards, in a minimum number of steps.

Note that blue player's goal is also to collect all awards, in a minimum number of steps.

Score:

  • The score is being initial with 50.
  • The score will be calculated using the following:
    • Colliding with the blue ball = 0.
    • Each time you move = -2.
    • Remain in place = -1
    • Reaching an award = 2
    • Reaching the goal of the game = 100.

Good luck!
